# Fastwind

Fastwind is a NLTE, unified stellar atmosphere/spectrum synthesis code that allows to calculate the structure of hot massive star atmospheres incl. line-driven wind and their emergent spectra in a robust and fast way. This repository comprises

**Fastwind version 10 and 11 (inicalc)**, and four additional directories related to data of background elements.<br>
**At first,** download only *inicalc* and check *manual_Fastwind.pdf* contained in the main directory.

The following directories can be downloaded **later**, if at all:<br>
**inicalc_ATOMDAT_NEW_old**: if comparison with previous versions (until v10.7.x/v11.7.x) aimed at.<br>
**inicalc_ATOMDAT_2016_new**: if extended atomic structure (mostly for low ionization stages) shall be
tested/implemented<br>
**make_ATOMDAT_NEW_new**: if data base (bg elements) *ATOMDAT_NEW_new* shall be updated<br>
**make_ATOMDAT_2016_new**: if data base (bg elements) A*TOMDAT_2016_new* shall be updated<br>
